# Evolutionary Project Seed (v4.0)

This is a "Universal Seed" for bootstrapping software projects as "Living Territories." It uses the Gemini CLI Evolutionary Engine (5 Skills + 1 Constitution).

## The v4.0 Philosophy: Metabolic Elegance
A system that burns more than it extracts will starve. We use the **Metabolic Ledger** to track costs and the **Architecture Trail** to ensure simplicity. We no longer just "Survive"; we "Thrive" by finding the most economical path to the truth.

## The Evolutionary Loop (v4.0):
1.  **`chaos-explorer` (Scout):** Proactively maps WAF/Rate-limits in the Sandbox.
2.  **`scar-archivist` (Historian):** Records failures with visual and DOM evidence.
3.  **`reaction-simulator` (Efficiency):** Fixes blocks using the **Inference Hierarchy** (Tier 1 > Tier 2 > Tier 3).
4.  **`manifold-compressor` (Architect):** Performs **Economic & Elegance Audits**. Refactors logic to reduce token burn and technical debt. Ensures **Hash Parity**.
5.  **`genesis-surveyor` (Founder):** Starts new projects with a "Golden Baseline."

## Key Files & Mandates:
- `GEMINI.md`: The Constitution (v4.0). Includes Sentinel Protocols and Economic Mandates.
- `METABOLIC_LEDGER.md`: Tracks token and time burn rate.
- `ARCHITECTURE_TRAIL.md`: The ADR (Architectural Decision Record) and Retrospective log.
- `SCHEMA_DEPENDENCY_FOREST.md`: The North Star for data integrity.
- `.gemini/fossil_library/`: The hierarchical library of environmental wisdom.
- `data/golden/`: The sacred baseline for zero-regression testing.
